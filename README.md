## 🎯 สิ่งที่ผู้เรียนจะได้รับ

- เข้าใจพื้นฐาน Generative AI, LLM, Transformer, Reasoning Model และการเลือกใช้โมเดลให้เหมาะกับงาน
- ยกระดับจาก **Prompt Engineering** สู่ **Context Engineering** เพื่อสั่งงาน AI ให้ออกแบบ เขียน วิเคราะห์ และแก้ไขโค้ดได้แม่นยำ
- ใช้ **AI Coding Agent** ในกระบวนการพัฒนาจริง ตั้งแต่วางแผน เขียนโค้ด รีแฟกเตอร์ เขียนเทสต์ ไปจนถึง Code Review
- เชื่อมต่อ AI กับเครื่องมือและข้อมูลองค์กรผ่าน **MCP (Model Context Protocol)**
- สร้างแอปที่ขับเคลื่อนด้วย AI ด้วย **RAG, Tool/Function Calling และ AI Agent**
- ประเมิน ทดสอบ และนำระบบ AI ขึ้น Production ด้วยแนวทาง **LLMOps, Evaluation, Observability และ Guardrails**
- ปิดท้ายด้วย **Capstone Project** สร้างระบบ AI จริงตั้งแต่ออกแบบจนถึง Deploy
