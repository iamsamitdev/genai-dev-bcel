# Extra Workshop — อ่าน CSV หลายไฟล์มาเติมลง Excel หลักที่ผูกสูตรไว้แล้ว

> โมดูลเสริม (Extra) สำหรับหลักสูตร Claude Cowork & Code — วันที่ 1
> ต่อยอดจาก Module 4a (สร้าง Excel Dashboard) โดยเน้นสถานการณ์ที่พบบ่อยจริงในงานประจำ:
> "มีไฟล์ Excel แม่แบบที่ผูกสูตรไว้แล้ว แล้วต้องเอาข้อมูลดิบจากหลายไฟล์ CSV มาเติมโดยไม่ทำสูตรพัง"

---

## 1. สถานการณ์จำลอง (Scenario)

บริษัทมีสินค้าคงคลังกระจายเก็บใน **3 คลังสินค้า** ได้แก่ กรุงเทพ เชียงใหม่ และขอนแก่น
ทุกสิ้นเดือน แต่ละคลังจะ Export ไฟล์ CSV รายงานจำนวนสินค้าคงเหลือ (Stock Count) ส่งมาให้ส่วนกลาง

ส่วนกลางมีไฟล์ **Excel หลัก (`stock_master.xlsx`)** ที่ผูกสูตรคำนวณไว้เรียบร้อยแล้ว
งานที่ต้องทำทุกเดือนคือ นำจำนวนคงเหลือจากไฟล์ CSV ทั้ง 3 คลัง มาเติมลงในคอลัมน์ที่กำหนด
โดยจับคู่ด้วย **รหัสสินค้า (SKU)** แล้วปล่อยให้สูตรคำนวณมูลค่าและสถานะให้อัตโนมัติ

---

## 2. ไฟล์ในโมดูลนี้

| ไฟล์ | ประเภท | รายละเอียด |
|------|--------|-----------|
| `stock_bangkok.csv` | ข้อมูลดิบ | สต็อกคลังกรุงเทพ (12 SKU) |
| `stock_chiangmai.csv` | ข้อมูลดิบ | สต็อกคลังเชียงใหม่ (10 SKU — ขาด SKU-1009, SKU-1012) |
| `stock_khonkaen.csv` | ข้อมูลดิบ | สต็อกคลังขอนแก่น (10 SKU — ขาด SKU-1002, SKU-1010) |
| `stock_master.xlsx` | **ไฟล์หลัก (โจทย์)** | มี Master สินค้า + สูตร + Dashboard — คอลัมน์คลัง (สีเหลือง) ยังว่าง |
| `stock_master_filled.xlsx` | **เฉลย** | ไฟล์หลักที่เติมข้อมูลครบแล้ว ใช้เทียบผลลัพธ์ |

> **จุดสอนสำคัญ:** ทั้ง 3 ไฟล์ CSV มี **ลำดับแถวไม่ตรงกัน** และ **บางคลังไม่มีสินค้าบาง SKU**
> ถ้าเติมด้วยมือแบบ Copy-Paste ตรง ๆ ตัวเลขจะเลื่อนผิดแถวทันที — จึงต้องจับคู่ด้วย SKU เท่านั้น

## 3. โครงสร้างไฟล์ Excel หลัก (`stock_master.xlsx`)

### Sheet `Master`

| คอลัมน์ | หัวตาราง | ที่มา |
|--------|---------|-------|
| A | รหัสสินค้า (SKU) | มีอยู่แล้ว (ใช้เป็น key จับคู่) |
| B | ชื่อสินค้า | มีอยู่แล้ว |
| C | หมวดหมู่ | มีอยู่แล้ว |
| D | หน่วย | มีอยู่แล้ว |
| E | ต้นทุน/หน่วย (บาท) | มีอยู่แล้ว |
| F | จุดสั่งซื้อ (ROP) | มีอยู่แล้ว |
| **G** | **คงเหลือ-กรุงเทพ** | ⬅️ **เติมจาก `stock_bangkok.csv`** |
| **H** | **คงเหลือ-เชียงใหม่** | ⬅️ **เติมจาก `stock_chiangmai.csv`** |
| **I** | **คงเหลือ-ขอนแก่น** | ⬅️ **เติมจาก `stock_khonkaen.csv`** |
| J | รวมคงเหลือ | 🔒 สูตร `=SUM(Gn:In)` |
| K | มูลค่าคงเหลือ (บาท) | 🔒 สูตร `=Jn*En` |
| L | สถานะ | 🔒 สูตร `=IF(Jn<=Fn,"⚠️ ต้องสั่งเพิ่ม","✅ ปกติ")` |

> คอลัมน์ **G, H, I** ถูกไฮไลต์สีเหลืองในไฟล์ = "ช่องที่ต้องเติม"
> คอลัมน์ **J, K, L** ผูกสูตรไว้แล้ว **ห้ามพิมพ์ทับ** ให้ปล่อยสูตรทำงานเอง

### Sheet `Dashboard`
สรุปอัตโนมัติจาก Master: มูลค่าสต็อกรวม, จำนวนคงเหลือรวม, จำนวน SKU, จำนวน SKU ต่ำกว่าจุดสั่งซื้อ, มูลค่าเฉลี่ยต่อ SKU และมูลค่าสต็อกแยกรายคลัง

## 4. Prompt สั่งงาน Claude Cowork (ใช้กรอบ Context + Task + Format + Constraint)

```
Context: ฉันมีไฟล์ Excel หลัก stock_master.xlsx (Sheet "Master") ที่ผูกสูตรไว้แล้วในคอลัมน์
J, K, L และมีไฟล์ CSV รายงานสต็อก 3 คลังในโฟลเดอร์เดียวกัน

Task: อ่านไฟล์ CSV ทั้ง 3 ไฟล์ แล้วเติม "จำนวนคงเหลือ" ลงในไฟล์หลัก โดยจับคู่ด้วยรหัสสินค้า (SKU):
  - stock_bangkok.csv   -> คอลัมน์ G (คงเหลือ-กรุงเทพ)
  - stock_chiangmai.csv -> คอลัมน์ H (คงเหลือ-เชียงใหม่)
  - stock_khonkaen.csv  -> คอลัมน์ I (คงเหลือ-ขอนแก่น)

Constraint:
  1. จับคู่ด้วย SKU เท่านั้น เพราะแต่ละ CSV เรียงลำดับไม่เหมือนกัน
  2. ถ้า CSV คลังไหนไม่มี SKU นั้น ให้ใส่ค่า 0 (คลังนั้นไม่มีสินค้า)
  3. ห้ามแก้คอลัมน์ J, K, L เด็ดขาด — เป็นสูตร ให้ปล่อยคำนวณเอง
  4. เขียนทับไฟล์ stock_master.xlsx โดยรักษาสูตรและรูปแบบเดิมไว้ทั้งหมด

Output: บันทึกไฟล์ stock_master.xlsx แล้วสรุปให้ฉันว่า มูลค่าสต็อกรวมเท่าไร
และมี SKU ใดบ้างที่ต่ำกว่าจุดสั่งซื้อ
```

## 5. เกณฑ์ตรวจผลลัพธ์ (Verify)

เทียบกับ `stock_master_filled.xlsx` — ควรได้ค่าเหล่านี้:

- มูลค่าสต็อกรวมทั้งหมด = **91,685 บาท**
- จำนวนคงเหลือรวมทุกคลัง = **869 หน่วย**
- จำนวน SKU ต่ำกว่าจุดสั่งซื้อ = **4 รายการ** ได้แก่ SKU-1002, SKU-1009, SKU-1010, SKU-1012
- มูลค่าแยกคลัง: กรุงเทพ 44,210 / เชียงใหม่ 24,580 / ขอนแก่น 22,895

> ✅ **จุด Human-in-the-Loop:** ก่อนเชื่อผลลัพธ์ ให้สุ่มตรวจ 2-3 SKU ว่าค่าในคอลัมน์ G–I ตรงกับ CSV จริง
> และตรวจว่าสูตรในคอลัมน์ J, K, L ยังเป็นสูตรอยู่ (ไม่ถูกพิมพ์ทับเป็นตัวเลขนิ่ง)
